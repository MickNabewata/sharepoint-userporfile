【ファイル構成】

	任意のフォルダ

		- ユーザー更新.bat	：処理実行時に起動するバッチファイル
		- ユーザー更新.ps1	：ユーザー更新.batから呼び出され、実際の処理を行うPowerShellファイル
		- CreateCred.bat	：処理実行アカウントをパスワードファイルに保存するために使うバッチファイル
		- userCred.xml		：処理実行アカウントを記録したパスワードファイル
		- ユーザー一覧.csv	：更新対象ユーザーの情報を記載したCSVファイル

【実行手順】

	1．ユーザー一覧の用意

		ユーザー一覧.csvを準備してください。
		このファイルに記載のデータに従ってユーザーが更新されます。

	2．CreateCred.batを実行

		ユーザー更新の実行アカウントでパスワードファイルを作成します。

	3．ユーザー更新.batの実行

		batを実行します。
